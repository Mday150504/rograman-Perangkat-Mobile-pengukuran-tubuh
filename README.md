# Kalkulator Indeks Massa Tubuh (IMT) - React Native

Aplikasi mobile sederhana untuk menghitung **Indeks Massa Tubuh (IMT)** berbasis **React Native**.  
Aplikasi ini dirancang agar mudah digunakan, dengan antarmuka bersih dan intuitif untuk mengukur kategori berat badan berdasarkan tinggi dan berat pengguna.

## Deskripsi Proyek

Proyek ini dibuat sebagai **Tugas Individu Pertemuan 3** oleh **M. Dai Misbah**.  
Aplikasi menerima input **berat badan (kg)** dan **tinggi badan (cm)**, kemudian menghitung IMT dan menampilkan kategori berat badan:

- Kurus
- Normal
- Berat Badan Berlebih
- Obesitas

## Fitur

- Input berat badan dan tinggi badan
- Kalkulasi IMT otomatis
- Menampilkan kategori hasil IMT
- Tampilan mobile-friendly dan interaktif
- Desain dengan warna kontras dan mudah dibaca

## Teknologi yang Digunakan

- **Frontend & Logic**: React Native
- **Bahasa Pemrograman**: JavaScript
- **Platform**: Android / iOS (melalui Expo atau React Native CLI)
